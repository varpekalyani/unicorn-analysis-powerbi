## Overview
This project presents an interactive Power BI dashboard analyzing
global unicorn startups across countries, industries, and investors.

## Key Highlights
- 1000+ unicorn companies analyzed
- Data from 40+ countries
- 15+ industry domains
- Average time to become unicorn: ~7 years
- Total funding analyzed: $500+ billion

## Dashboard Features
- Country and continent-wise unicorn distribution
- Industry-wise company analysis
- Year-wise growth of unicorn startups
- Top investors overview
- Valuation to funding ratio analysis
- Interactive slicers for filtering insights
